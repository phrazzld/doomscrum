# Brainrot Vibe Meter

Priority: P2 · Status: done · Estimate: S

## User
Operator comparing multiple generated PRD videos.

## Problem
Provider output can be technically valid but spiritually corporate.

## Goal
Record a simple human vibe rating on each render so bad models can be culled later.

## Acceptance Criteria
- Render detail allows rating a clip from cursed to corporate.
- Rating is stored as a local event.
- Rating does not mutate the source PRD.

## Risk
Humor quality is subjective and should not block the core artifact contract.
