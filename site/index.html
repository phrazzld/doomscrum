the gestures

swipe right dispatch a real agent, open a real PR
swipe left skip — spec stays untouched
swipe up skip, recorded durably
swipe down back to the previous spec
tap read the source spec — path + sha256

Arrow keys mirror every gesture. Space pauses. Enter opens the spec. The first implement-swipe against a repo always confirms first — Cook it / Cancel.

quickstart — under two minutes, no account

# macOS (Homebrew)
brew install phrazzld/doomscrum/doomscrum

doomscrum init      # scaffold doomscrum.toml
doomscrum doctor    # check your setup
doomscrum serve     # 127.0.0.1:4173 — sample videos play immediately

Bundled sample videos play with zero keys and zero config edits. Run doomscrum generate afterward to render your own repo's specs against real providers.

legal & safety, plainly stated

  • Videos are AI-generated, derived from backlog spec text. Not verified fact, not real people, not real events.
  • Real rendering costs money and sends spec-derived text to OpenRouter and fal.ai. DoomScrum quotes the cost first and enforces hard spend caps. The fixture provider is free and never leaves your machine.
  • Right swipes launch real agents that modify real code and open real pull requests — there is no sandbox theater beyond what your agent CLI provides.
  • MIT licensed. Full disclosure, data-egress enumeration, and the pre-launch legal checklist: docs/LEGAL.md.
